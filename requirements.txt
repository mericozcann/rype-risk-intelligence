# RYPE — Explainable Operational Risk Intelligence

RYPE (Risk Yield Propagation Engine) is a research prototype for explainable geopolitical supply-chain risk intelligence.

It connects externally grounded geopolitical risk signals with mechanism-driven D1–D4 risk propagation, explainability, counterfactual scenario analysis, resilience evaluation, and human-in-the-loop decision support.

## Scope

This is a research MVP and decision-support demonstrator, not a real-time industrial shipment monitoring product.

The current version uses externally grounded geopolitical and port-risk data layers, precomputed research experiment outputs, and a route-level operational risk inference layer for demonstration.

## Key Components

- Geopolitical risk grounding
- Port and UN/LOCODE entity layer
- D1–D4 mechanism-driven risk propagation
- Explainable risk decomposition
- Counterfactual route comparison
- Scenario stress laboratory
- Temporal and resilience analysis
- Human-in-the-loop decision support
- Decision intelligence dashboard

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
